== Description ==

Virealys est un thème WordPress révolutionnaire conçu pour le premier restaurant Slow Food immersif & évolutif. Il intègre un système de navigation spatiale par panneaux, une technologie d'anticipation prédictive, et une interface utilisateur entièrement adaptative.

**Fonctionnalités principales :**

* Navigation spatiale par panneaux plein écran (pas de défilement traditionnel)
* Dock prédictif adaptatif — les boutons se réordonnent selon les préférences de l'utilisateur
* Cursor light — lumière ambiante suivant la souris
* Éléments magnétiques — les boutons attirent le curseur à proximité
* Ripple au clic — onde néon à chaque clic
* Mode respiration au repos — le site pulse quand l'utilisateur est inactif
* Parallaxe par profondeur basée sur le curseur
* Personnalisation complète via le Customizer WordPress
* Support de l'éditeur de blocs avec palette de couleurs personnalisée

== Installation ==

1. Téléchargez le thème et décompressez-le dans le répertoire `/wp-content/themes/`
2. Activez le thème via le menu 'Apparence' dans WordPress
3. Configurez les options du thème dans Apparence > Personnaliser

== Structure des fichiers ==

* `style.css` — Métadonnées du thème
* `functions.php` — Fonctions et configuration
* `theme.json` — Configuration des blocs et styles
* `assets/css/` — Styles principaux
* `assets/js/` — Scripts adaptatifs
* `assets/fonts/` — Polices locales (optionnel)
* `templates/` — Modèles de page (Ambiance, Concept, Menus, Zones, Passeport, Contact)
* `parts/` — Fragments de templates réutilisables
* `patterns/` — Compositions de blocs
* `styles/` — Variations de styles

== Modèles de page disponibles ==

* **Ambiance** — Template universel pour toutes les ambiances. Utilise l'image mise en avant et l'extrait.
* **Le Concept** — Page concept du restaurant
* **Nos Menus** — Page des menus gastronomiques
* **Les Zones** — Page des 4 zones d'expérience
* **Passeport Virealys** — Programme de fidélité
* **Contact & Réservation** — Page de contact
